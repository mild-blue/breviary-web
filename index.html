How to install application on Android devices.


To be able to use Breviary Android application now, you have to allow to install application from unknown sources. It is because application is not available on Google Play Store yet.

  • Navigate to Setting > Security.
  • Check the option Unknown sources.
  • Tap OK on the prompt message.
  • Select Trust.
